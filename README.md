## Clone and Run Locally
1) Clone the repository
  ```
  git clone https://github.com/adigapranava/multiplayerGame.git
  ```
2) Go to the App
  ```
  cd multiPlayer
  ```
3) Go to Server
  ```
  cd Server
  ```
4) Run Server
  ```
  npm start
  ```
5) open local host `http://localhost:3001/`
